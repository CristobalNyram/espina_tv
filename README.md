## Steps to install


Install packages
```shell
composer update 
```
```shell
composer install 
```

Copy env

```shell
cp .env.example .env 
```

Generate key

```shell
php artisan key:generate 
```

Generate scripts 

```shell
npm run development 
```

**Client:** Vue js 3,Blade , Jquery
